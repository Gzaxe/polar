# 极化散射截面计算 —— 进度与笔记

本仓库用于跟踪 **极化散射截面（Polarized Scattering Cross Section）** 相关计算工作的进度，并保存所有研究过程中的笔记、文档和临时记录。

## 仓库定位

- **本仓库（GitHub）**：存储 **进度跟踪、笔记、会议记录、文献笔记** 等非代码类文档。  
  不包含可执行的计算脚本，会有一些原始数据或结果文件，因为数据处理有时候会在win下直接做去更方便的看结果。

- **另一个仓库（Gitee）**：专门用于 **Linux 环境下的 Fresco 计算**，包含所有输入文件、处理脚本和计算结果。  
尽量不重复两边的计算内容。
